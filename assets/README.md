Place the official crest image provided by the user here:

- Path: assets/crest.png
- Filename must be: crest.png
- Recommended format: PNG with transparent background (keep original resolution).

If you place `crest.png` in the `assets` folder, the site will display it in the header and the identity seal. If absent, a vector fallback `crest-fallback.svg` will be used automatically.
 
Add the homepage hero image (building photo):

- Path: assets/hero.jpg
- Filename must be: hero.jpg
- Recommended: JPEG, max width 2000px for fast loading. Keep original as backup.

Place the provided building photo into `assets/hero.jpg` and refresh `index.html` to see it as the hero background. If `hero.jpg` is missing, a dark gradient will be used instead.
